## About

Slope Finder helps skiers and snowboarders discover the best ski resorts based on their location and preferred travel dates. Simply enter where you're coming from and when you want to hit the slopes, and we'll show you the best options available.

## Features

- **Location-based search** - Find resorts near you or any starting point
- **Date filtering** - Check resort availability for your travel dates
- **Resort details** - View information about each ski resort
- **Responsive design** - Works seamlessly on desktop and mobile



## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm run dev
   ```

The app will be available at `http://localhost:3000`

If you want to deploy it, you may use the Dockerfile. It is currently setup for Google Cloud Run.
